# Enjoyce

A mobile app for the online travel agency, Enjoyce Travel and Tours.

## Getting Started
1. Install [Flutter](https://docs.flutter.dev/get-started/install?gad_source=1&gclid=CjwKCAiA0bWvBhBjEiwAtEsoW7Gg-xvAg71RxmgyOH5BK5zjwsEIoL0eLKaVcK9ZNdNwLn0m5dOeVBoCYswQAvD_BwE&gclsrc=aw.ds) (and other relevant SDKs and requirements)
2. Clone the repo with `gh repo clone HolyShaq/Enjoyce`
3. Run the app using `flutter run`

## Codebase Structure
- Put **assets** (i.e icons, svgs) in **lib/assets**
- Separate major **pages** (i.e log-in, homepage) and put them in **lib/pages**
- Extract major **components** (i.e buttons, input boxes) and put them in **lib/components**
